# Instructions
Using only the history.data file for now, make a graph showing the evolution of your star on the theoretical Hertzsprung-Russell diagram (log-L vs. log-Teff). Also show on that diagram some version of the full zero-age main sequence of stars. You may lookup a theoretical line, calculate your own, or convert observed star values to these quantities. Cite any sources for this. 

Choose some points of interest on the evolutionary path, label them, and using the information in the history file, describe what is happening in your star at those times. Include descriptions in major changes of nuclear burning, radius, etc. Note the age of the star at those times, and explain the time between them, if possible. 

Make two plots of quantities from the history file versus time that show something interesting.

You may share the work on this, but each of you should be able to make the graphs. It is ok if you both submit the same file(s) if you both work on them.
